### ⚖️ Calculadora de IMC - Windows Forms



Este é um projeto desenvolvido durante os meu estudos de **C#** e **Lógica de Programação**. A aplicação utiliza o framework **Windows Forms** para calcular o Índice de Massa Corporal (IMC) de um utilizador com base no peso e altura fornecidos.



#### 🛠️ Tecnologias Utilizadas



* **Linguagem:** C#
* **Framework:** .NET (Windows Forms)
* **IDE:** Visual Studio



#### 📈 O que utilizei neste projeto



* Manipulação de eventos de interface.
* Tratamento de exceções e validação de `inputs`.
* Lógica condicional estruturada (`if/else if/else`).
* Formatação de strings numéricas (ex: `ToString("F2")`).
* Botão para limpar campos e resetar a aplicação rapidamente.
